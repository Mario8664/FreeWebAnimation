FreeWebAnimation 用户操作指南

这份文档面向使用者，按当前编辑器代码整理演示界面、参数编辑、动画片段、场景时间轴和视频导出的实际操作。

1. 启动与关闭

  1. 双击或运行 启动演示.bat。
  2. 启动器会打开一个独立的 Chrome/Edge 演示窗口。
  3. 关闭这个演示窗口后，对应项目的 Vite dev server 会自动停止。

如果默认端口被其它项目占用，启动器会自动尝试下一个可用端口。需要严格使用指定端口时运行 启动演示.bat -StrictPort。

如果直接运行 npm run dev，浏览器关闭后不会自动停止服务，需要在终端手动停止。

2. 主界面区域

左侧参数区

显示全局参数和当前页面参数。PBR 页面会出现 PBR Global Parameters，普通页面会出现自己的场景参数面板。

中间预览区

显示当前页面画面。底部 W / H 可以修改渲染宽高，常用导出尺寸是 1920 x 1080。

右侧动画区

包含 Clips 和 Timeline 页签。上方还会放坐标轴等调试工具。

底部 Prev / Next 用于切换页面。Three.js 页面默认显示坐标轴；如果页面是 3D 画面叠加 DOM/SVG/Canvas，仍然按 3D 页面显示坐标轴。

3. 参数编辑器

  • 点击参数面板标题可以折叠或展开整个面板。
  • 点击参数分组标题可以折叠或展开该组参数。
  • 数字参数可以直接输入，也可以按住参数名左右拖动微调。
  • 颜色参数使用颜色选择器；范围参数使用滑条；复选框控制开关；下拉框用于模式选择。
  • 可动画参数旁边有一个小菱形关键帧按钮。没有接入动画的参数不会显示这个按钮。

保存规则

  • 场景参数修改后，底部 保存 会变成 保存*。
  • 点击底部 保存 保存当前页面全部参数。
  • 某个分组有改动时，分组标题旁会出现单组保存按钮，只保存该组相关参数。
  • 动画 Preview 开启时不能保存场景参数；需要先关闭 Preview。
  • 全局 PBR 参数也需要手动保存，但它不参与动画关键帧。

复制与粘贴分组

  • 场景参数分组标题旁的复制按钮会把该组参数写入剪贴板。
  • 粘贴按钮会从剪贴板读取同结构参数。
  • 参数名、数量或类型不匹配时，粘贴会被拒绝。
  • 全局 PBR 参数面板不提供分组复制/粘贴。

参数 JSON 是手动保存；动画 Clip 和 Timeline 的改动会在操作后自动保存到 public/assets/animations/。

4. Clips 动画片段

Clips 页签用于制作可复用动画片段。一个 Clip 记录若干参数在不同时间的关键帧。

创建与管理

  1. 点击 + Clip 创建片段。
  2. 点击列表中的片段名称选择当前 Clip。
  3. 在 Name 输入框修改名称，按 Enter 或让输入框失焦后生效。
  4. 点击 Delete 删除片段。

时间与预览

  • Time 输入框控制当前 Clip 时间。
  • 在 Record Timeline 空白处点击或拖动，可以移动当前 Clip 播放头。
  • 鼠标滚轮缩放 Record Timeline 视图，< / > 平移视图，+ / - 缩放视图。
  • 拖播放头或拖关键帧时，靠近已有关键帧会吸附；按住 Ctrl 会放大吸附范围。
  • 播放按钮会播放或暂停当前 Clip。播放结束后再次播放会从头开始。
  • 勾选 Preview 后，当前 Clip 采样结果会应用到画面；关闭后恢复到已保存的基础参数。

添加与删除关键帧

  • 先选择一个 Clip，再把 Time 移到目标时间。
  • 点击参数旁的小菱形按钮，会为该参数在当前 Time 添加关键帧。
  • 如果当前 Time 已经有该参数关键帧，再点击小菱形按钮会删除该参数关键帧。
  • 如果只选中了一个关键帧，并且该时间点已经有这个参数的关键帧，小菱形按钮会优先删除选中时间点的这一帧。
  • 想把新参数补到某个已选关键帧上，需要先把 Time 调到那个时间点。
  • 选中一个或多个关键帧后，点击 ⌫ 会删除这些时间点上的所有参数关键帧。

5. 关键帧选择与编辑

  • 点击关键帧点选择单个关键帧。
  • 按住 Shift，在 Record Timeline 上拖出框选区域，可以多选关键帧。
  • 按住 Shift 从某个关键帧点开始拖，也会进入框选，不会移动该关键帧。
  • 拖动一个已选中的关键帧时，如果当前有多个已选关键帧，会一起移动。
  • 拖动关键帧靠到已有关键帧时间点时会合并同时间的参数关键帧；按住 Ctrl 更容易吸附到已有关键帧。

Keyframe 面板

  • 右侧 Keyframe 区域会列出选中时间点共同拥有的参数。
  • 插值方式有 Auto、Linear、Step。
  • 多选关键帧时，如果同一参数的插值不同，会显示 Mixed；改成其它值会批量应用。
  • 如果某个参数关键帧是该参数轨道的第一帧，插值下拉框会禁用，因为没有进入这一帧的上一段过渡。

6. 实时录制

  1. 在 Clips 页签选择一个 Clip，并把 Time 移到录制起点。
  2. 点击圆点录制按钮开始录制；按钮变成红色方块状态。
  3. 录制期间，时间会自动向前走。调整可动画参数时，会把变化写入关键帧。
  4. 再次点击录制按钮会停止录制。切换页签、移动时间、打开预览或执行 Fit 时，也会先停止录制。

录制开始时会记录一份参数快照。某个参数第一次发生变化时，如果它原本没有轨道，系统会补一个起点关键帧，让动画从录制起点平滑开始。

7. Fit Cleanup

Fit 用于压缩实时录制造成的密集关键帧，主要适合摄像机轨迹或连续数值参数。

  1. 在 Clips 页签至少选中两个关键帧。
  2. 勾选 Fit 打开 Fit Cleanup 浮层。
  3. 在 X/Y/Z 轴下拉框中选择数值参数路径；不需要的轴选择 None。
  4. 同一个参数路径只能占用一个轴。重新选择时，其它轴上相同路径会被清掉。
  5. 调整 Simplify。数值越高，保留的关键帧越少。
  6. 勾选 Uniform speed 会按路径长度重新分布保留关键帧。
  7. 点击 Fit 应用清理；点击 Restore 恢复本次 Fit 前的关键帧。

Fit 按钮需要“至少两个关键帧”和“至少一个有效轴参数”才可用。浮层顶部可以拖动，用来避开画面。

8. Scene Timeline

Timeline 页签用于把多个 Clip 组合成最终导出的视频时间轴。

Source Clips

  • 把 Source Clips 中的 Clip 拖到下方轨道，会生成一个 Clip 实例。
  • 双击某个 Source Clip，会把它添加到第一条轨道的当前场景时间。

场景播放头与视图

  • Timeline 页签里的 Time 输入框控制场景时间轴时间。
  • 在 Scene Timeline 空白处点击或拖动，可以移动场景播放头。
  • 鼠标滚轮缩放 Scene Timeline 视图，< / > 平移视图，+ / - 缩放视图。
  • 播放按钮会播放或暂停整个场景时间轴。播放到末尾后再次播放会从头开始。

Clip 实例

  • 点击实例块可以选中它。
  • 拖动实例块可以改变开始时间和所在轨道。
  • 拖动实例块左右边缘可以改变长度。
  • 移动、缩放实例时按住 Ctrl，会吸附到其它实例边缘或源 Clip 的播放长度。

Instance 面板

选项 作用
Start 实例在场景时间轴中的开始时间。
Length 实例在场景时间轴中占用的时长。
Speed 源 Clip 的播放速度。数值越大，源 Clip 内容播放越快。
Loop 源 Clip 播完后循环填满实例长度。
Reverse 反向采样源 Clip。
Post: Hold 实例结束后保留最后一帧参数，直到被后续活动实例或更高优先级轨道覆盖。
Post: None 实例结束后不保留参数。
Delete 删除当前选中的 Clip 实例。

混合与轨道优先级

  • 同一轨道中两个实例重叠时，重叠区域会在两个 Clip 的采样结果之间平滑混合。
  • 尽量避免同一轨道三段以上同时重叠；当前采样只混合排序后的前两个活动实例。
  • 活动实例会覆盖同名参数的 Hold 结果。
  • 多条轨道出现同名参数时，编号更靠前的轨道优先级更高：Track 1 高于 Track 2，Track 2 高于 Track 3。

时间标记

  • 在 Scene Timeline 空白区域双击可以添加时间标记。
  • 拖动标记可以移动位置；按住 Ctrl 会吸附到其它标记或 Clip 实例边缘。
  • 右键标记可以删除。

Timeline Base

Timeline Base 开关显示在 Timeline 页签。开启后，回到 Clips 预览单个 Clip 或录制时，会先按当前场景时间轴采样一遍，再把当前 Clip 叠加上去。它适合检查某个片段放进整体时间轴后的效果。

9. 视频导出

导出依赖保存后的 Timeline 文件。只有 public/assets/animations/<sceneId>.timeline.json 里存在正时长实例的页面，才会被列为可导出场景。

  • 交互式导出：导出视频.bat
  • 查看可导出场景：导出视频.bat --list
  • 导出指定场景：导出视频.bat --scene pbr-showcase
  • 批量导出所有可导出场景：导出视频.bat --batch 或 导出视频.bat --all
  • 命令行方式：npm run export:video -- --scene pbr-showcase

导出时会启动临时 Vite 服务，用浏览器逐帧截图，再调用 ffmpeg 合成 mp4。默认输出目录是 output/videos，默认中间帧目录是 .export-frames。

参数 说明
--width / --height 输出分辨率，默认 1920 x 1080。
--fps 输出帧率，默认 60。
--reserve 时间轴前后各保留的停留秒数，默认 2。
--out 视频输出目录。
--frames 临时 PNG 帧目录。
--keep-frames 编码后保留 PNG 帧，方便检查导出问题。

导出需要可用浏览器和 ffmpeg。ffmpeg 可以放在 tools/ffmpeg/bin/ffmpeg.exe，也可以加入系统 PATH。

10. 自由摄像机与坐标轴

自由摄像机

只有页面接入自由摄像机组件时才生效。按住鼠标右键拖动可以旋转视角；按住右键时使用 W/A/S/D 平移，Q/E 上下移动，滚轮调整移动速度。松开右键或窗口失焦后，移动输入会停止。

坐标轴

Three.js 页面默认有坐标轴调试层；纯 2D 页面没有。3D 页面叠加 2D UI 时，仍然按 Three.js 页面处理，所以会显示坐标轴。

11. 常见问题

为什么保存按钮不能点？

当前处于动画 Preview 状态。先取消 Preview，再保存场景参数。

为什么导出列表里没有我的页面？

页面需要有保存后的 timeline 文件，并且时间轴里至少有一个正时长 Clip 实例。

为什么多选关键帧没反应？

需要按住 Shift 后再在 Record Timeline 上拖框。普通拖动播放头只会移动时间。

为什么修改参数后关闭 Preview 又变回去了？

Preview 是临时动画采样状态。要修改基础场景参数，请关闭 Preview 后再调整并保存。
